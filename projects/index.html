Projects

Tap4s
Tap4s
A security type system for the P4 programming language, TAP4S employs a lightweight interval analysis combined with security labels to guarantee the noninterference of P4 programs.
p4 type system sdn

DiVerT
DiVerT
A prototype tool that relies on type-based dependency analysis and symbolic tuples to track the dependencies of different program execution paths, and verify them against the disjunctive security policy of the underlying database.
type-based analysis disjunctive policies database-backed programs

DynCoVer
DynCoVer
A prototype tool to verify program security under dynamic policies. DynCoVer employs symbolic execution and Z3 solver to symbolically execute a Java program and check its security against the active policy at each observable output.
java symbolic execution dynamic policies

Je
Je
A programming language model to develop secure enclave applications. Je uses high-level abstractions to partition a Java program into enclave and non-enclave parts, and a security type system to guarantee the robustness of the enclave part
java type system trusted execution environments
